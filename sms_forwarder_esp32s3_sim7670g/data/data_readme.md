# 配置示例文件

本目录包含配置文件的示例，仅供参考。实际运行时，系统会在SPIFFS中自动生成配置文件。

## 文件说明

### config.json
系统配置示例，包含：
- WiFi连接配置
- 推送平台配置（Bark、Server酱、Telegram等）
- 电池管理配置
- 网络配置
- 短信过滤配置
- 系统设置

### sms.json
短信存储格式示例，包含：
- 短信记录结构
- ID管理
- 转发状态

## 注意事项

- 这些文件**不会**被上传到ESP32设备
- 系统启动时会自动在SPIFFS中创建配置文件
- 如果需要预设配置，请修改代码中的默认值
- 配置文件存储在ESP32的SPIFFS文件系统中，路径为 `/config.json` 和 `/sms.json`
